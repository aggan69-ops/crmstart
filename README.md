Södertörns Team CRM - production ready starter

Det här är versionen för riktiga användare.

Nytt:
- förberett för Supabase Auth
- .env.example för riktiga hemligheter
- Fortnox server-side helper
- SQL-schema för riktiga tabeller
- pipeline, offerter, tasks, samtal, synk-kö
- widgetstyrning
- audit log

Start:
1. Ladda upp filerna till repo
2. Deploya på Vercel
3. Behåll NEXT_PUBLIC_DEMO_MODE=true först
4. När du är redo: lägg in Supabase-nycklar och sätt NEXT_PUBLIC_DEMO_MODE=false
